Why use sherlock?

A good test is a automated test, but it's not always possible to automate everything.

Ever project have an suite of regression tests. Some projects have outside hardware depencies and some have tests that are so abstract that is really tricky to automate.

For those cases, until today, you had to pay expensive tools per user and month or use outdate tools that are heavy and hard to use and extend.

Sherlock is a option for those scenarios, as it's a opensource and a free tool that you can use and extend as you need.

Sherlock is free and will always be free!
